## Overview

The evidence repository is a centralized location for storing compliance documents, screenshots, and other evidence.

## Supported File Types

- PDF documents
- Images (PNG, JPG)
- Word documents
- Excel spreadsheets
- Text files

## Organizing Evidence

<Steps>
  <Step title="Upload">
    Click **"Upload Evidence"** and select your file.
  </Step>
  <Step title="Categorize">
    Assign to a framework requirement or program.
  </Step>
  <Step title="Tag">
    Add tags for easy searching.
  </Step>
  <Step title="Link">
    Link evidence to specific controls or requirements.
  </Step>
</Steps>
